## 技术栈

### 后端技术

- Spring框架
- Spring MVC
- MyBatis

### 前端技术

- Bootstrap
- jQuery
- Layui
- JSTL
- JSTP

- MySQL

## 开发工具

- IntelliJ IDEA
- Eclipse

## 功能模块

1. 前台展示
   - 商品浏览
   - 商品搜索
   - 购物车管理
   - 订单提交与支付
   - 用户评论
2. 后台管理系统
   - 商品管理
   - 订单管理
   - 用户管理
   - 数据统计与分析

## 系统角色

- 普通用户：浏览商品、购物、评价等。
- 管理员：负责商品管理、订单处理、用户管理等。

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL 5.7 或更高版本
- 服务器：Tomcat 8.5 或更高版本
- Java版本：JDK 1.8 或更高版本

## 部署步骤

1. 准备运行环境，安装并配置好 MySQL 数据库和 Tomcat 服务器。
2. 导入数据库文件，建立数据库。
3. 部署应用到 Tomcat 服务器。
4. 启动 Tomcat 服务器并运行系统。
